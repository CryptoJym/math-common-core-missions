### ðŸ’¡ ðŸš€ Mission Strategy: Step-by-Step
*(Use this guide to help you complete the instructions above)*
1. Draw a chart with 3 rows: **X > (Greater)**, **X = (Equal)**, **X < (Less)**.
2. Roll your dice (e.g., you roll a 6). Write '6' in the blanks.
3. **Greater Than:** Find an object longer than 6 inches.
4. **Equal:** Find an object exactly 6 inches.
5. **Less Than:** Find an object shorter than 6 inches.
6. Repeat for 3 different rolls!
