1. **Part 1 (Addition):** Equation is X + A = 12.
2. Roll dice to find 'A' (e.g., 4).
3. Solve: X + 4 = 12. (Answer: X must be 8). Check with blocks!
4. **Part 2 (Multiplication):** Equation is AX = B.
5. Use the chart provided in the instructions. If A=3 and B=12...
6. Split 12 blocks into 3 piles. How many in each pile? That is X.
